WhereYouGo is an unofficial client for Wherigo Geocaching. It uses offline vector maps, alternatively Locus can be used to display maps.

Important note:
This is a dedicated legacy version of WhereYouGo for Android 4.1 - 4.4 devices to maintain basic functionality!
Future versions will no longer be available on your device! Please consider using a device with a newer Android versions to be able to use WhereYouGo in future.

Cartridges for the game can be downloaded at
www.wherigo.com

Download cartridges for Pocket PC Device. Other versions are not tested (but it's possible they will work).

Happy caching and nice weather!
